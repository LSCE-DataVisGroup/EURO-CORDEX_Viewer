Foreword
This web application aims at providing a quick view on the mean projected climate evolution in each european country and in some other neighboring countries. It displays expected changes, together with associated uncertainties as provided by the EURO-CORDEX multi-model ensemble of simulations. It also aims to provide a quick intercomparison among model projections in order for modelers to better understand their model's behaviour.

Context

  • CORDEX (Coordinated Regional Climate Downscaling Experiment) web site - wcrp-cordex.ipsl.jussieu.fr.

Technical descriptions

  • The interactive map is coded with Leaflet, an Open-Source JavaScript Library for Mobile-Friendly Interactive Maps - leafletjs.com.
  • The chart is coded with HighCharts, a JavaScript Charting Library - www.highcharts.com.
  • European countries displayed and used to compute integrations have been processed from boundaries provided by www.naturalearthdata.com/downloads/50m-cultural-vectors.
  • Shortnames for countries respect the ISO_3166-1_alpha-3 convention - en.wikipedia.org/wiki/ISO_3166-1_alpha-3.
  • Note that countries are colored randomly each time you reload the page.
  • Masks and all processed integrations are available from the Thredds catalog : http://webportals.ipsl.jussieu.fr/thredds/catalog/EUROCORDEX/catalog.html. The Time Series Viewer takes advantages from the NCSS (NetCDF Subset Service) to fetch data to the browser.
  • When multiple choices are made both on models entry and times entry, the different dash line styles are applied on the models entry.
